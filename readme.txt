# BloemKracht Fitness App

Een vriendelijke fitness app met korte 10-minuten oefeningen voor 50+, speciaal ontworpen voor mensen met knieblessures en met een rustgevende hippie/flower power/Indiase stijl.

## Over de App

BloemKracht is een persoonlijke trainingsapplicatie die speciaal is ontworpen voor mensen van 50+ die korte maar effectieve trainingen willen doen met aandacht voor knieblessures. De app heeft een zachte, aanmoedigende aanpak in plaats van een drill sergeant-stijl, en biedt oefeningen in een prettige, rustgevende omgeving.

### Hoofdkenmerken

- **Korte oefensessies**: Alle oefeningen zijn ontworpen om maximaal 10 minuten te duren, waardoor ze gemakkelijk in een drukke dag passen
- **Knie-vriendelijke oefeningen**: Speciale aandacht voor veilige been- en knieoefeningen
- **Hippie/Indiase stijl**: Rustgevende kleuren, bloemmotieven en mandala's
- **Motiverend speelelement**: Een bloeiende mandala die groeit naarmate je vaker oefent
- **Zachte, aanmoedigende feedback**: Vriendelijke begeleiding tijdens de oefeningen

## Beschikbare Sessies

1. **Ochtendenergie** (5 min) - Een zachte reeks om je dag te beginnen
2. **Knieversterker** (8 min) - Gericht op veilige oefeningen voor je knie
3. **Avondrust** (6 min) - Ontspannende bewegingen voor bedtijd

## Technische Details

Deze app is gebouwd met:

- React Native / Expo
- Verschillende componenten voor de interface-elementen
- Expo Speech voor de aanmoedigende voice-overs
- Animaties voor de visuele elementen

## Installatie

1. Zorg ervoor dat je Node.js en npm hebt geïnstalleerd
2. Clone de repository:
```
git clone https://github.com/jouw-username/bloemkracht-fitness-app.git
```
3. Navigeer naar de projectmap:
```
cd bloemkracht-fitness-app
```
4. Installeer de afhankelijkheden:
```
npm install
```
5. Start de app:
```
npm start
```
